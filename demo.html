Janapada

embed tutorial

How to embed

1

Prepare your data files

An evolution.json (generated in step 3) and one GeoJSON file per census year, served alongside your HTML.

my-project/ ├── evolution.json ← generated below ├── boundaries-1951.geojson ├── boundaries-1961.geojson └── index.html

Each GeoJSON needs a property for the unit name (e.g. district_name) and optionally a parent (e.g. state_name).

2

Write a transitions CSV

One row per lineage. Columns are {year}-{level1} / {year}-{level2} pairs. Empty cells mean the unit did not exist that year. Splits have multiple rows from one source; merges have multiple rows to one destination.

# transitions.csv
1951-state,1951-district,1961-state,1961-district
Bihar,Hazaribagh,Bihar,Hazaribagh
Bihar,Hazaribagh,Bihar,Hazaribagh
Bihar,Hazaribagh,Jharkhand,Chatra
Bihar,Hazaribagh,Jharkhand,Koderma
3

Generate evolution.json

Option A — Node.js (recommended)

# install once
npm install -g janapada

# run
janapada janapada.yml

Option B — R

install.packages(c("yaml", "jsonlite"))
source("make_evolution.R")

make_evolution.R produces identical output to the Node CLI, including fuzzy GeoJSON matching and BFS chain coloring.

Config file (janapada.yml)

name: "India Districts"
levels:
  level1: state
  level2: district
years: [1951, 1961, 1971, 1981]
transitions: ./transitions.csv
sources:
  - year: 1951
    geojson: ./boundaries-1951.geojson
    key: district_name
    parent_key: state_name
fuzzy_match: true
output: ./evolution.json
4

Add the script tag

<script src="https://saketlab.github.io/janapada/lib/janapada.iife.js"></script>

React, D3, and styles are all bundled in. No build tooling needed.

~700 KB uncompressed, ~212 KB gzipped.
5

Call Janapada.init()

All-years grid view

Janapada.init('#map', {
  evolutionFile: 'evolution.json',
  geojsonSources: [
    { year: 1951,
      url: 'boundaries-1951.geojson',
      key: 'district_name',
      parentKey: 'state_name' },
    // ... one entry per year
  ],
});

Open on the district tab

Janapada.init('#map', {
  evolutionFile: 'evolution.json',
  geojsonSources: [ /* ... */ ],
  initialMode:    'district',
  defaultDistrict: 'Hazaribagh',
});

Unmount

Janapada.destroy('#map');

All options

evolutionFile    string   required
geojsonSources   array    required
darkMode         boolean  false
initialMode      string   'grid' | 'single' | 'district'
defaultDistrict  string   pre-loaded district name
noOriginLabel    string   legend label for untraced units
